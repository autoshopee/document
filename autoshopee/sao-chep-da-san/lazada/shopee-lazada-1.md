# Lazada => Lazada

{% hint style="info" %}
* Bạn cần Xem và hiểu hướng dẫn Copy từ Shopee => Lazada trước khi bắt đầu.
* Hệ thống chỉ hỗ trợ Copy từ Shop Lazada chính chủ (tức là Shop gốc đã thêm vào hệ thống)
{% endhint %}
